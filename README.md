# Profit

A daily intelligence briefing for ambitious software engineers who want to improve their technical depth, understand markets, and navigate career uncertainty.

Heavy compute, RAG operations, and multi-agent loops run overnight, so the user receives an instantly rendered briefing at launch with zero wait.

## Features

### Tech Learning Engine

Combined OS, DBMS, DSA, and System Design tracks delivered through a progressive-disclosure accordion.

- **The Reality Check** — a single high-caliber interview question.
- **The Key Insight** — a punchy 3-minute read delivering the core architectural concept.
- **Deep-Dive [Expandable]** — low-level system pathways, memory buffers, and deep technical context.
- **Code & Architecture [Expandable]** — executable pseudo-code, database schemas, or visual layouts.
- **User Learning Memory** — logs concept completions, quiz performance, and weak points, then injects scenario variations in future nightly generations to reinforce gaps.
- **Content Metadata Scoring** — each lesson carries a Difficulty Score, Novelty Level, and Interview Frequency rating.

### Portfolio Tracker

Plaid-connected market tracking that maps macro causality onto a user's active holdings.

- **Plaid Brokerage Sync** — read-only Plaid Investments tokens; the backend reads active holdings nightly to build the ticker list.
- **Manual Entry Fallback** — a clean text input for users who prefer not to authenticate a brokerage.
- **Macro Causal Analysis** — isolates catalysts such as Federal Reserve yields' effect on software equity discounting.
- **7-Day Historical Time Machine** — collapsible drawers cache a timeline of daily post-mortems for any held asset.

### International Student Module

Tactical intelligence for users with the `is_international_student` flag enabled.

- Tracks federal policy adjustments, visa processing updates (F-1, OPT, H-1B), and regional tech employment flows.

### Core Platform Enhancements

- **Granular Feedback Loop** — every content block supports Useful, Not Useful, Too Basic, and Too Advanced feedback.
- **Explicit Citation Layer** — market and visa claims anchor to primary sources (e.g., SEC EDGAR, Fed releases) via structured citation blocks.


## AI Pipeline

A 6-tier nightly agent workflow:

1. Extraction
2. Clustering
3. Contextualization
4. Narrative Synthesis
5. Citation Mapping
6. Quality Validation — a programmatic gatekeeper that blocks formatting errors, hallucinations, and missing components.

Primary feeds (SEC EDGAR filings, Alpha Vantage news, Department of State Visa Bulletins) are prioritized over generic RSS, and regulatory disclosures are treated as neutral data vectors rather than price indicators.
